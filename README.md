## Description:

This RESTful API returns a json object with aphorisms


### Basic usage


```bash 
curl aphorismcookie.herokuapp.com/
```

also we have a mirrored api hosted at

```bash
curl https://sage.af/api/fortune/
```

Response

```json
"data": {
  "message": "Everything will now come your way.",
}
```
